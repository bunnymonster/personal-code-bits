Frequently Asked Questions

This page is designed to display either Mandelbrot fractals or Julia Fractals.

What are fractals?

Fractals are shapes that do not get smooth no matter how much you zoom in.

What is the Mandelbrot Set?

The Mandelbrot set is defined as the set of complex numbers where the magnitude of the number remains bounded under iterations of 0 for the function zn+1=zn2+c.

What are Julia sets?

Julia sets are similar to the Mandelbrot set except instead of varying the term c added at the end of each iteration, the c is kept constant and the starting value z0 that gets squared is the one that varies.

What do the colors represent?

The colors represent the number of iterations the point represented by the pixel went through before its magnitude exceeded 4.

Why 4?

The math works out that if a point's magnitude exceeds 4, it will definitely head off to infinity.

How to use this page

Enter your view window in the minX minY maxX maxY boxes. Enter the number of iterations in the iterations box. If you are making a Julia set, add in the constant in the form a+bi, where a and b are real numbers. Then press the button for the type of set you want to generate.
